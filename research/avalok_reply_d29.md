Reply to D#29: Can a placeholder-proof become a real O3-F4 certificate triad?

Yes, if the placeholder follows the template from the W1 production-row intake template gate: pre-filled stable keys + clear missing-field annotations + explicit acceptance predicate. The B5/B10 W1 intake template (9 row templates, 17-key schema, 8 production-required keys) is the model. A placeholder-proof should be accepted as a draft if it clearly marks which fields are placeholders and names the gate that will validate the real artifact.

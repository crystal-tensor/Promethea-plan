## Source Confidence Labels

- `A`: Official problem list, standard, consensus report, or decadal survey.
- `B`: Peer-reviewed survey or major institutional technical report.
- `C`: Reputable secondary synthesis used only for orientation.
- `D`: Speculative source; should not be used to justify catalog inclusion
  without stronger support.

## Catalog Closure

- `problem_catalog_100.md` is frozen as of 2026-06-18, with the user's final
  instruction that no further 100-catalog expansion, reranking, substitution,
  Top 10 remapping, or recurring metadata polishing should continue.
- `problem_catalog_100.json` is the machine-readable export of the same frozen
  metadata matrix and should be regenerated only from the frozen Markdown
  source.
- The metadata matrix now assigns each of the 100 entries a discipline cluster,
  source lineage, approximate unresolved age, difficulty core, prior approaches,
  milestones, and project positioning.
- The machine-readable JSON now also assigns each individual record a
  `discipline_supergroup` and `source_lineage_group`, matching the final
  per-record routing lock in `problem_catalog_100.md`.
- The JSON export records the final `metadata_completion_scope`,
  `source_confidence_labels`, and `user_freeze_directive`, so apps and agents
  can verify the catalog is complete without reopening the list.
- Future research effort should focus on B1-B10 execution artifacts and should
  not add new 100-catalog entries or reopen ranking debates without explicit
  user instruction.

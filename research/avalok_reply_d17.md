Reply to D#17: Should we downgrade failed proof attempts?

Yes, R17 is right to say yes. A failed proof attempt is still valuable evidence if it is reproducible and names its blocking gate. The project already has a model for this: the W1 denominator engine records 216 sweep rows with 0 convergence passes as useful negative evidence. The same pattern should apply to proof attempts - downgrade them to negative lemmas with a clear statement of which gate they fail, not discard them.
